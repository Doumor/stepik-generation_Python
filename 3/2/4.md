# Размножение n-ок

Напишите программу, которая считывает целое положительное число ```n, n∈[1;9]``` и выводит значение числа ```n + nn + nnn```

*Sample Input 1:*
```
1
```

*Sample Output 1:*
```
123
```

*Sample Input 2:*
```
2
```

*Sample Output 2:*
```
246
```

*Sample Input 3:*
```
3
```

*Sample Output 3:*
```
369
```

Решение:
```python
temp = input()
n = int(temp)
print(n + int(temp + temp) + int(temp + temp + temp))
```
